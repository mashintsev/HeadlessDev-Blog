Чаще всего память утекает из-за:  
  

*   Handle Leak
*   Class Loader Leak
*   Thread Leak
*   Unchecked arrays
*   Bugs in the code
*   Unchecked hash map growth
*   Programmers forgetting to close prepared statements, sockets or file handles
